Day 1 - July 21, 2026
File: Day1Functions.java
Topic: Methods - static methods, parameters, return values, methods
       calling methods
Notes: Java methods must declare a return type (int, void, String, etc.)
       or "void" if nothing is returned. Java methods can only return
       ONE value (unlike Python's multiple return). Called "methods"
       not "functions" since they live inside a class. Default parameter
       values are not supported directly in Java.

Day 2 - July 22, 2026
File: Day2Arrays.java
Topic: Arrays - creating, indexing, length, looping (for-each), sorting
Notes: Java arrays have a FIXED size once created - cannot add/remove
       elements like Python lists. Must declare one type only.
       .length is a property (no parentheses), unlike Python's len().
       Sorting requires java.util.Arrays.sort(). For resizable
       collections, Java uses ArrayList instead (separate topic).
